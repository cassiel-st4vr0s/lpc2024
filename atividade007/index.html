About the game

Up to the Sky is an educational platformer game designed to enhance typing skills. The player can leap from platform to platform, reaching each one by pressing the corresponding keys. As the game progresses, it introduces more keys and challenges, requiring players to type faster and more accurately. This engaging gameplay not only improves typing speed and accuracy but also makes learning to type fun and interactive.

Game Details

How the gameplay works:
Each platform has a set of keys(e.g. "FJ" ), the player should type the correct keys, so the character will jump on the platform, in the beggining is simple as that.
After some pregression, levels will have new challenges like: Projectiles coming in the player's direction, each one will have a set of keys, and it will be the player's mission type the corresponding keys to defend himself from the projectiles.

Progression and levels structure:
One of our objectives is involve all the main keys of the "Left Hand" and the "Right Hand", but at the same time assure the game insn't too big and as balanced as possible. See the details below:

Levels
